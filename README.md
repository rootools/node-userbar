# Install
```
npm install userbar
```
# Usage
```
var userbar = require('userbar');

var options = {
    ltext: 'Left Text',
    rtext: 'Right Text',
    ltextxoffset: 0,      // Left text X offset
    rtextxoffset: 40};    // Right text X offset

userbar.createUserbar(options);
```

After this out.png will be created in your`s project root folder.
